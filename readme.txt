O que foi feito até agora:

Criação da interface simples do front-end que realiza um fetch na rota do back-end, o back-end busca os dados no banco do prisma e mostra os dados na tela em uma tabela estilizada.


Como rodar o front e o back:

front: npm run dev direto na pasta: library-prisma.

back: ir para o back-end e rodar o comando: npx ts-node app.ts
